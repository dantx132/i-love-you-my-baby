# 📖 Livro de Memórias - Dante & Duda

Um site romântico interativo com:
- 📸 Galeria de fotos em formato de livro
- 🎁 Livro de presentes (Dante para Duda & Duda para Dante)
- 📅 Datas especiais
- 📱 100% responsivo para celular
- 💻 Sem backend - totalmente client-side

## 🚀 Como usar

1. Clonar o repositório
2. Abrir `index.html` no navegador
3. Adicionar fotos nas pastas:
   - `galeria-fotos/` - Galeria de fotos
   - `presentes-fotos/presentes de dante pra duda/` - Presentes de Dante
   - `presentes-fotos/presentes de duda pro dante/` - Presentes de Duda

4. Regenerar listas (opcional):
   ```bash
   python regenerate_presentes_list.py
   python regenerate_gallery_list.py
   ```

## 📝 Adicionar fotos

Basta adicionar imagens nas pastas correspondentes e recarregar o site!

## 🌐 Deploy no GitHub Pages

1. Copiar todos os arquivos da pasta `dist/`
2. Colocar em um repositório GitHub
3. Habilitar GitHub Pages nas configurações
4. Site estará disponível em `https://seu-usuario.github.io/repo-name`

## 📱 Responsivo

O site funciona perfeitamente em:
- 💻 Desktop
- 📱 Tablets
- 📲 Smartphones
